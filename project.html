Correlating Tweets with Weather - California, New Jersey

Info

In this project, we collected tweets on selected topics from states of New Jersey, California for 3 weeks using Python Tweepy. Weather data was obtained from Wunderground website. Using Pandas, we tried to see if the weather correlates with the activities or tweets posted about certain activities. The activites we chose to pursue were - working out, going out, eating, shopping. In California, due to same weather throughout the duration, there was no much correlation but New Jersey showed a promising correlation. We then tried to apply machine learning algorithms such as Linear Regression to predict the number of tweets on a particular activity based on weather. The project report, code are available in GitHub if you wish to dig deeper into the details.

Predicting the readmission of diabetic patients

Info

A dataset on diabetic patients was obtained from UCI Machine Learning Repository. We performed data cleaning on the data to remove columns with missing data or fill gaps in data. Using Weka software, we tried a number of machine learning algorithms such as Random Forest, NaiveBayes, Linear Regression, K-means clustering to predict if a patient is readmitted into a hospital based on his health information.

Math Performance based on playing Action Video Games

Info

We took a dataset that has information on the performance of math exam taken by Non-STEM students before and after playing an action video game. We conducted statistical analysis (ANOVA) to check whether there is any difference if the students played an action video game. Out of curiosity, we also wanted to find out if gender played any role in the difference of scores. To find out we performed ANOVA and Linear Regression Analysis on the data set using SPSS. We concluded based on the results obtained in SPSS.
